    笑点解析

    • 低调的黑客

      [00:00] BiliBili YouTube
      “低调的黑客”这个词来自25年10月份作用的一个新梗，这类梗的内容大多以“你知道生气的黑客有多恐怖吗”开头，通常视频内容搭配一些无厘头毫无意义的片段

    • 低客的黑调

      [00:11] BiliBili YouTube
      由“低调的黑客”重组为“低客的黑调”，其汉语拼音为“Dī kè de hēi diào”，同时可逆空耳谐音为“dick的黑屌”。

    • 看完blue的是这个

      [00:16] BiliBili YouTube
      “blue”是“不撸”的逆空耳，此句话实意为“看完不撸的是这个”，“撸”这里指男性自慰性行为

    • 注意图中没有领带

      [00:19] BiliBili YouTube 领带幻视为男性的生殖器官

    • 注意黑客不一定是低调，而黑调一定是低客

      [00:23] BiliBili YouTube
      前半句用于衬托后半句的出现，“而黑调一定是低客”是“而黑屌一定是dick”的逆空耳

    • 你现在一定在看b站，你一定看到了这视频，你一定看到了tietou

      [00:30] BiliBili YouTube
    • “b站”是中国人对BiliBili视频平台的简称，本视频源在BiliBili
    • “tietou”是“重音teto”中的teto的音译
      所以本句为废话文学

    • 91-78=13

      [00:36] BiliBili YouTube
    • “91”在这里指“91传媒/91制片厂”是拍黄色视频黄色综艺的组织
    • “78”是“鸡巴”的谐音，是拍黄色视频黄色综艺的组织
    • “13”在这里形似“B”，“B”是“逼”（女性生殖器官）的谐音

    • 青春伤痛一定是遗憾，低调的黑客看过魔丸

      [00:36] BiliBili YouTube
    • “青春伤痛一定是遗憾”表面上说的是遗憾，实际上暗指青春的伤痛可能来源于性经历
    • “低调的黑客”在此处无意义，“魔丸”是中国电影《哪吒之魔童闹海》中的主角"哪吒"的民间称谓

    • 部分词义解释：

      • 购袋来了

        “购袋”指性别是沃尔玛购物袋，是一个起源于英文互联网论坛和社交媒体（如 Reddit、Tumblr 等）的流行梗， 通常用作对“身份政治”（Identity Politics）和非二元性别认同（Non-binary Gender Identities）的一种戏谑、讽刺或嘲弄

      • 武机来了

        “武机”指性别是武装直升机，同上

      • 哈基米南，北绿豆哦

        “哈基米南北绿豆”是“はちみつなめると”的空耳， 其中“はちみつ”（哈基米）是赛马娘第二季第12集中约10分20秒处 角色东海帝皇在去购买蜂蜜饮品时哼唱的内容， 而“なめると”（南北绿豆）的出现是因为“哈基米南北绿豆”近期经常被很多创作者拿去给各种歌曲填词

      • 350234，往日种种，可有话说，再无话说，苦命鸳鸯

      • 350234是指在禁漫天堂中一个作品它的编号是350234这个作品由于剧情较为猎奇而出圈， 大致剧情是讲吕布和董卓的爱情故事
      • “往日种种，可有话说，再无话说，苦命鸳鸯”都是350234中部分图片的台词

      • 前面忘了，中间忘了，后面忘了

        这段话通常用于糊弄的回答问题然后阐述一个观点
        例如：前面忘了，中间忘了，后面忘了，总之“苦命鸳鸯”
        大意就是不管怎么回事总之类似苦命鸳鸯的场景

    • 我是奶龙，我才是奶龙

      [01:19] BiliBili YouTube
      “奶龙”是子供向动画《奶龙》系列动画及其衍生作品中的虚构恐龙， 由于奶龙行为较为诡异形似唐氏综合症而出众
      “我是奶龙，我才是奶龙”出自《奶龙》第126集中真假奶龙的桥段
