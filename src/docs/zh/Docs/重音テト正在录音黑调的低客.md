  - 部分词义解释：
    - #### 购袋来了
      “购袋”指性别是沃尔玛购物袋，是一个起源于英文互联网论坛和社交媒体（如 Reddit、Tumblr 等）的流行梗，
      通常用作对“身份政治”（Identity Politics）和非二元性别认同（Non-binary Gender Identities）的一种戏谑、讽刺或嘲弄
    - #### 武机来了<br>
      “武机”指性别是武装直升机，同上
    - #### 哈基米南，北绿豆哦
      “哈基米南北绿豆”是“はちみつなめると”的空耳，
      其中“はちみつ”（哈基米）是[赛马娘第二季](https://zh.wikipedia.org/wiki/%E8%B3%BD%E9%A6%AC%E5%A8%98_Pretty_Derby_(%E5%8B%95%E7%95%AB))第12集中约10分20秒处
      角色[东海帝皇](https://zh.wikipedia.org/wiki/%E8%B3%BD%E9%A6%AC%E5%A8%98_Pretty_Derby_(%E5%8B%95%E7%95%AB))在去购买蜂蜜饮品时哼唱的内容，
      而“なめると”（南北绿豆）的出现是因为“哈基米南北绿豆”近期经常被很多创作者拿去给各种歌曲填词
    - #### 350234，往日种种，可有话说，再无话说，苦命鸳鸯
      1. [350234](https://18comic.vip/album/350234/)是指在[禁漫天堂](https://18comic.vip/)中一个作品它的编号是350234这个作品由于剧情较为猎奇而出圈，
      大致剧情是讲[吕布](https://zh.wikipedia.org/wiki/%E5%90%95%E5%B8%83)和[董卓](https://zh.wikipedia.org/wiki/%E8%91%A3%E5%8D%93)的爱情故事
      2. “往日种种，可有话说，再无话说，苦命鸳鸯”都是[350234](https://18comic.vip/album/350234)中部分图片的台词
      {#parse3}
    - #### 前面忘了，中间忘了，后面忘了
      这段话通常用于糊弄的回答问题然后阐述一个观点<br>
      例如：前面忘了，中间忘了，后面忘了，总之“[苦命鸳鸯](#parse3)”<br>
      大意就是不管怎么回事总之类似[苦命鸳鸯](#parse3)的场景
