# 欢迎来到梗指南 {#jjj}

欢迎来到这个项目！这里是为视频专门整理的“笑点解释指南”，帮助你快速理解视频里的梗和幽默点。

## 项目说明
- 文档里会详细说明视频里的笑点、梗出处以及相关背景。
- 适合想快速掌握梗的人或复习视频中的趣味内容。

## 注意
- 每个文档都按照视频顺序整理。
- 如果遇到不懂的梗，可以使用左侧导航快速查找相关解释。
- 欢迎提出新的梗或者改进意见，让指南更完善。

## 基础概念
1. 谐音：本土语言听成另一种本土语言
2. 空耳：外国语言听成本土语言
3. 逆空耳：本土语言听成外国语言
4. 空耳谐音：包含1和2
5. 逆空耳谐音：包含1和3
6. 形似：把一个事物看作另一个事物
