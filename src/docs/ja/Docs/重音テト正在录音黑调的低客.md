    - #### 购袋来了
    「购袋」は、性別がウォルマートのショッピングバッグであることを指します。これは、英語圏のインターネットフォーラムやソーシャルメディア（Reddit、Tumblrなど）を起源とする流行のミームで、
    通常、「アイデンティティ政治」（Identity Politics）やノンバイナリーな性自認（Non-binary Gender Identities）に対する皮肉、風刺、または嘲笑として使われます。

       「哈基米南北绿豆」は「はちみつなめると」の空耳です。
       「はちみつ」（哈基米）は、[ウマ娘 プリティーダービー Season 2](https://ja.wikipedia.org/wiki/%E3%82%A6%E3%83%9E%E5%A8%98_%E3%83%97%E3%83%AA%E3%83%86%E3%82%A3%E3%83%BC%E3%83%80%E3%83%BC%E3%83%93%E3%83%BC_Season_2)の第12話、約10分20秒あたりで、
